# CSS3 TP

This folder contains a small set of HTML/CSS practice exercises.

## Files

- `index.html`: main page with links to all exercises
- `ex1.html` to `ex8.html`: individual exercises

## How to use

Open `index.html` in your browser to access all exercises.

You can also open any exercise file directly, for example:

- `ex5.html` for the responsive layout exercise

## Purpose

These files are meant for learning and practicing:

- basic HTML structure
- CSS styling
- responsive design with media queries
